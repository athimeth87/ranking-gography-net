# GOGRAPHY Ranking — Future Revenue Models

> วิเคราะห์จากจุดแข็งของ platform + ตลาดที่เข้าถึงได้

---

## จุดแข็งที่ platform นี้มี (asset ที่มีอยู่แล้ว)

ก่อนคิดถึง model ใหม่ ต้องรู้ว่าตอนนี้มีอะไรที่มีมูลค่าอยู่แล้ว:

| Asset | ทำไมถึงมีมูลค่า |
|---|---|
| Community ช่างภาพ + นักเดินทาง | กำลังซื้อสูง, passionate, loyal |
| Ranking algorithm (Pulse Score) | ระบบวัดคุณภาพภาพที่เป็น objective |
| Photography content คุณภาพสูง | visual content ที่หายากในตลาดไทย |
| Data พฤติกรรม + ความชอบ | รู้ว่าคนชอบภาพแบบไหน, ถ่ายที่ไหน, ใช้กล้องอะไร |
| Brand GOGRAPHY | ความน่าเชื่อถือในตลาด photography travel |
| Voyageur network | คนที่เคยไปทริปด้วยกัน — strong bond |

## Model ที่น่าสนใจสำหรับอนาคต

---

### Model A — Photographer Booking Platform
**"Airbnb สำหรับช่างภาพ"**

แทนที่จะแค่โชว์ผลงาน — ให้ช่างภาพที่ verified แล้วรับงานจากลูกค้าผ่านเว็บได้เลย

**ทำงานอย่างไร:**
- ลูกค้า (แบรนด์, โรงแรม, บริษัทท่องเที่ยว) โพสต์ job brief
- ช่างภาพที่ Rank สูงหรือ Ambassador รับ brief
- GOGRAPHY รับ commission 15–20% จาก deal

**ทำไมถึงได้ผล:**
- แบรนด์ไทยอยากได้ช่างภาพที่มี portfolio จริงและผ่านการ verify แล้ว
- Pulse Score กลายเป็น "credibility score" ที่ลูกค้า B2B เชื่อได้
- GOGRAPHY ไม่ต้องลงทุนเพิ่มมากเพราะ talent pool มีอยู่แล้ว

**รายได้โดยประมาณ:** deal ละ 5,000–50,000 บาท × 20% commission

---

### Model B — White-label Contest Platform (SaaS)
**"ขาย platform นี้ให้คนอื่น"**

องค์กร, โรงแรม, tourism board, สายการบิน ต้องการ run photo contest ของตัวเอง แต่ไม่มี tech — GOGRAPHY ขาย platform เป็น service

**ตัวอย่างลูกค้า:**
- TAT (การท่องเที่ยวแห่งประเทศไทย) — contest "ถ่ายภาพประเทศไทย"
- โรงแรมใหญ่ — "Best photo at our resort"
- สายการบิน — "ถ่ายภาพจากความสูง 35,000 ฟุต"
- จังหวัด / เทศบาล — photo contest ส่งเสริมท่องเที่ยว

**สิ่งที่ขาย:**
- Platform พร้อม ranking, voting, leaderboard
- Branding ของลูกค้า (white-label)
- รายงานผล + export ข้อมูล

**ราคาแนะนำ:** 50,000–200,000 บาท/contest หรือ 15,000 บาท/เดือน แบบ subscription

**ทำไมถึงได้ผล:**
- โค้ดมีอยู่แล้ว — cost of goods ต่ำมาก
- ตลาด B2G (ภาครัฐ) และ B2B มีงบ contest สูง แต่ไม่มี tech
- Pulse Score algorithm เป็น differentiator ที่คู่แข่งไม่มี

---

### Model C — Data & Insights (B2B)
**"ขายข้อมูล anonymized ให้แบรนด์และ tourism"**

platform สะสมข้อมูลว่า:
- สถานที่ไหนถูกถ่ายมากที่สุด
- กล้อง/เลนส์อะไรได้ Pulse Score สูงสุด
- ช่วงเวลาไหนมี engagement สูง
- ประเภทภาพไหน (Landscape/Portrait/BW) ที่คนชอบ

**ลูกค้าที่น่าสนใจ:**
- บริษัทกล้อง — ต้องการรู้ว่า gear ไหนผลิต top-ranking photos จริงๆ
- Tourism board — ต้องการรู้ว่า location ไหน inspire นักเดินทาง
- นิตยสาร + media — trend report เรื่อง photography ในไทย

**รูปแบบ:**
- Annual Industry Report (PDF) — ขายราคา 5,000–20,000 บาท/ฉบับ
- Custom research — ราคา project-based
- API access สำหรับ developer — subscription รายเดือน

---

### Model D — Photography Education Ecosystem
**"มากกว่า masterclass — เป็น school"**

ต่อยอดจาก Ambassador program ที่มีอยู่:

| รูปแบบ | รายละเอียด | ราคา |
|---|---|---|
| Online course | Ambassador สอน technique เฉพาะทาง | 1,500–4,900 บาท |
| Live workshop | จัดใน Bangkok + location จริง | 3,000–8,000 บาท |
| Mentorship program | ช่างภาพ Top 10 รับ mentor ผู้เรียน 1:1 ต่อเดือน | 2,500 บาท/เดือน |
| Annual photography summit | event ใหญ่ปีละครั้ง — talks, portfolio review, awards | 1,500–5,000 บาท/ticket |

**revenue split แนะนำ:** GOGRAPHY 30% / ช่างภาพ 70%  
**ทำไมถึงได้ผล:** ช่างภาพใน platform มีผู้ติดตามอยู่แล้ว การขายผ่าน GOGRAPHY ได้ audience ที่ qualified

---

### Model E — Photography Location Guides (Digital Products)
**"ขาย knowledge ที่ community สร้างขึ้น"**

ข้อมูลใน platform รู้ว่า:
- ภาพ top-ranking ถ่ายที่ไหน, เวลาไหน, ด้วยกล้องอะไร
- ช่างภาพระดับ Ambassador ไปที่ไหนบ้าง

**สินค้า:**
- **Location Guide PDF** — "50 จุดถ่ายภาพในเชียงใหม่ ฉบับ GOGRAPHY" ราคา 299–499 บาท
- **Seasonal Calendar** — "ถ่ายภาพอะไร ที่ไหน เดือนไหน" ราคา 199 บาท/ปี
- **Interactive Map** — premium feature ใน app/web — ดู location pin ของรูป top-ranked

**ทำไมถึงได้ผล:**
- content นี้ community สร้างให้ฟรีอยู่แล้ว — GOGRAPHY แค่ curate และ package
- ต้นทุนต่ำมาก, scale ได้ไม่จำกัด

### Model F — Gear Affiliate Program (Structured)
**"รายได้ passive จากสิ่งที่มีอยู่แล้ว"**

ทุกภาพใน platform มีข้อมูล EXIF (กล้อง, เลนส์) อยู่แล้ว

**ทำอย่างไร:**
- เพิ่มลิงก์ affiliate ใต้ EXIF data ในหน้า photo detail
- เช่น "ถ่ายด้วย Sony A7R V → [ดูราคา]" (ลิงก์ไป JIB, Lazada, หรือ Sony Thailand)
- Affiliate commission ปกติ 2–5% ต่อยอดขาย

**ทำไมถึงน่าสนใจ:**
- ไม่ต้องสร้างอะไรใหม่มาก — แค่เพิ่ม affiliate link ใน UI ที่มีอยู่
- ผู้ใช้เห็น gear ที่ "ผ่านการพิสูจน์" โดยภาพ top-ranking — conversion rate สูงกว่า banner โฆษณาทั่วไป

---

### Model G — Exclusive Community Membership (Discord/Forum)
**"ขาย access ถึง community ชั้นใน"**

| Tier | สิทธิ์ | ราคา |
|---|---|---|
| Free | ดูรูป, โหวต, comment | ฟรี |
| Member | อัปโหลดรูป, portfolio page | 290 บาท/เดือน |
| Pro | Community forum, early access feature, portfolio analytics | 590 บาท/เดือน |
| Voyageur | ทุกอย่าง + ส่งเข้า Voyageur contest + trip priority | ราคา trip-linked |

**ทำไมถึงได้ผล:**
- recurring revenue ที่คาดได้
- community ที่ "pay to join" มักจะ serious กว่า → content quality สูงกว่า → ดึงดูด member ใหม่ได้ดีกว่า

---

### Model H — Corporate Photography Services
**"ต่อสาย brand กับช่างภาพ community โดยตรง"**

บริษัทใหญ่ต้องการภาพสำหรับ:
- Annual report
- Campaign marketing
- Social media content
- Hotel / resort photography

GOGRAPHY เป็น **creative agency** ที่มี talent pool verified อยู่แล้ว

**รูปแบบ:**
- Retainer package — แบรนด์จ่ายรายเดือนสำหรับ X ภาพ/เดือน
- Project-based — quote ต่อ project
- Revenue split กับช่างภาพ 60/40 หรือ 70/30

    │              │           │           │
    ▼              ▼           ▼           ▼
  Voyageur      Gear        Booking     White-label
  Membership  Affiliate    Platform      SaaS
  + Education  + Location  + Corporate  + Data &
               Guides      Services      Insights

**เหตุผล:**
- เริ่มจากสิ่งที่ใช้ asset ที่มีอยู่แล้ว (community, content, EXIF data)
- สร้าง recurring revenue ก่อน เพื่อให้มีกระแสเงินสดพัฒนา model ที่ใหญ่กว่า
- White-label SaaS เป็นเป้าหมายระยะยาว เพราะ scalable ที่สุด แต่ต้องใช้เวลาพัฒนา

## คำถามเชิงกลยุทธ์ที่ต้องตอบ

1. **GOGRAPHY อยากเป็นอะไร?** — Community platform, Creative agency, หรือ SaaS company? แต่ละทางต้องการ resource ต่างกัน
2. **ใคร run ธุรกิจนี้?** — White-label และ Corporate services ต้องการ sales team จริงๆ
3. **ข้อมูล user ได้รับ consent ไหม?** — ก่อนขาย Data & Insights ต้องแน่ใจว่า Terms of Service ครอบคลุม
4. **จะ balance ระหว่าง free community กับ paid features อย่างไร?** — ถ้า gate มากเกินไป community ตาย, ถ้าน้อยเกินไปทำเงินไม่ได้
